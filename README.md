# Osher Calisthenics - Sprint 1 Home

עדכון ממוקד למסך הבית בלבד.

מה כלול:
- Home חדש בסגנון אפליקציה.
- מפת שרירים קדמית/אחורית.
- נקודות לחיצה כחולות ומדויקות יותר.
- פאנל תרגילים שנפתח רק לאחר בחירת שריר.
- כרטיס תרגיל מרכזי עם תמונה, וידאו placeholder ופעולה.
- התאמה למובייל ו־RTL מלא.

איך לעדכן:
1. החלף בפרויקט שלך את הקבצים והתיקיות מתוך ZIP זה.
2. אל תעלה node_modules, .next או out.
3. במחשב הרץ: npm run build
4. אם הכל עובר, העלה ל־GitHub. Vercel יעדכן אוטומטית.
